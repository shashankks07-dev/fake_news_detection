# 📰 Fake News Detection System with Explainable AI

An interactive Fake News Detection Web Application built using Natural Language Processing (NLP) and Machine Learning, featuring multi-model comparison, explainability (LIME), and text analytics.

The system allows users to input news articles and instantly classify them as REAL or FAKE, along with insights into why the prediction was made.

---

## 🚀 Features

- 🧠 Multiple ML Models
  - Logistic Regression  
  - Naive Bayes  
  - Random Forest  

- 🔍 Real-time Prediction  
- 💡 Explainable AI (LIME)  
- 📊 Text Analytics  
- 📝 User-Friendly Interface  
- ⚙️ Model Selection Panel  

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- Scikit-learn  
- NLTK  
- LIME  
- Pandas, NumPy  

---

## ⚙️ How It Works

1. User enters a news article  
2. Text preprocessing (cleaning, stopwords removal)  
3. Feature extraction using TF-IDF  
4. Model predicts REAL or FAKE  
5. Optional LIME explanation and analytics  

---

## 📊 Dataset

- Fake/Real News datasets (Kaggle + Reuters-style structured data)
